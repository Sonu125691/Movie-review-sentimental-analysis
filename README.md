# Movie-review-sentimental-analysis
This project predicts the movie review is **Positive** or **Negative** using IMDB movie review dataset containing 5000 movie reviews. LSTM has been used in this project.
**Preprocessing:** Removed html words, Tokenization, Padding, Embedding.

## ▶️ Run the Project on Your Computer
**To run this project locally, use the following commands in your terminal:**

pip install -r requirements.txt

streamlit run app.py

**These commands will install the required libraries and open the app in your browser (running locally on your computer).**
